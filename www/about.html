Why ADR Bot? The Problem

📧

Decisions Lost in Slack

Architecture decisions get buried in channels and threads. No tracking, no structured record.

⏳

No Audit Trail

Who decided what, and when? Without event sourcing, the context behind decisions evaporates.

📄

ADRs Rot in Repos

Markdown files in git go stale. Nobody updates them because the friction is too high.

🔍

Hard to Find Decisions

Searching across repos and channels for "why did we choose X?" is painful and incomplete.

Transactional Outbox

Side effects queued in the same transaction. pg_cron sweeps, pg_net delivers. Zero lost notifications.

Git Export

Export ADRs as markdown PRs to your Git repository. Automatic branch creation and PR filing.

Full-Text Search

tsvector with GIN index plus ILIKE fallback. Search across titles, context, and decisions.

Block Kit Messages

Rich Slack messages with status badges, all populated fields, and inline action buttons.

Rate Limiting

Per-team rate limiting prevents abuse. 30 requests per minute with automatic recovery.

Dead Letter Queue

Failed outbox deliveries are tracked, retried, and eventually dead-lettered for investigation.
